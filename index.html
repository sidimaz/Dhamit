Règles du Dhamié

🎯 Aperçu

Le Dhamié se joue sur un plateau 9×9 avec 80 pièces (40 par joueur) placées sur les intersections. Tradition : bâtons dorés et crottes de chameau séchées sur sable.

♟ Pièces

  • Kalb (soldat) : avance en avant (ou diagonale avant si sur ligne diagonale). Capture dans toutes les directions.
  • Sultan ★ : atteint la dernière ligne adverse → se déplace et capture loin dans toutes les directions.

⚔️ Règles de base

  • Warda : premier coup obligatoire — bâtons : F4→E5 / crottes : D6→E5
  • Capture multiple : si vous pouvez enchaîner plusieurs captures, vous devez continuer
  • Annulation ↩ : demandez à annuler le dernier coup — l'adversaire doit accepter. Il faut jouer au moins 1 coup avant de pouvoir redemander.

⚡ Soufla — Mode automatique (ON)

  • Si vous ne capturez pas alors que vous pouviez, l'adversaire récupère la pièce fautive
  • Si vous capturez avec une pièce alors qu'une autre pouvait capturer plus, cette autre pièce est retirée
  • Si vous arrêtez une chaîne de captures alors que vous pouviez continuer, la pièce qui s'est arrêtée est retirée
  • Règle du nombre égal : si plusieurs pièces peuvent capturer le même nombre, vous êtes libre de choisir

🚨 Soufla — Mode manuel (OFF)

Quand la Soufla est désactivée, le bouton 🚨 Réclamer apparaît. Aucune détection automatique — c'est aux joueurs d'observer et de réclamer.

  • Comment réclamer :
    1. Appuyez sur 🚨 Réclamer
    2. Double-tapez sur la pièce fautive
    3. L'adversaire confirme avec le bouton ✅
    4. La pièce est retirée ✓
  • Chaîne incomplète : si un joueur s'arrête de capturer, un timer de 10 secondes démarre. L'adversaire peut alors double-taper sur la pièce qui s'est arrêtée pour réclamer la Soufla.
  • Règle du nombre égal : la Soufla ne s'applique que si une pièce pouvait capturer plus que celle jouée.

🏆 Fin de partie

  • Victoire : éliminer toutes les pièces adverses ou les bloquer
  • Nul (Koud) : Sultan contre Sultan

🎯 نظرة عامة

ظامت رقعة 9×9 بـ80 قطعة (40 لكل لاعب) على تقاطعات الخطوط. تقليد موريتاني أصيل: عصي ذهبية وبراز جمل مجفف على الرمل.

♟ القطع

  • الكلت: يتحرك للأمام (أو قطرياً إن كان على خط قطري). يقتل في كل الاتجاهات.
  • السلطان ★: يصل الصف الأخير → يتحرك ويقتل بعيداً في كل الاتجاهات.

⚔️ القواعد الأساسية

  • الوردة: الحركة الأولى الإجبارية — عصي: F4←E5 / براز: D6←E5
  • القتل المتسلسل: إذا أمكن الاستمرار في القتل يجب المتابعة
  • التراجع ↩: اطلب إلغاء آخر نقلة — يجب أن يوافق الخصم.

🏆 نهاية اللعبة

  • الغلب: القضاء على قطع الخصم أو تجميدها
  • الكود: سلطان ضد سلطان = تعادل
